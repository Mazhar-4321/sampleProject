CSS Stands For Cascading Style Sheets
It is used to give styles to a web page
We can change the height width
We can give background color text color and text alignment to the html tags
There are three ways to include css in our html programming
1)internal CSS
In internal CSS , we define style tag in the head tag
In the style tag we can access the elements in three ways
a) Using id : We use # symbol to access the element by its id . Every id needs to be unique
b) Using class name : We use . symbol to access the elements by its class name . A class need not be unique
c) Using element name : If we dont use any symbol then this type of access is called element name access
2) inline CSS
In inline css , we give style as an attribute to the element
3) External CSS
In external css , we include the file which is available at different location in the project
